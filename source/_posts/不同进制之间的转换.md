# 二进制和十进制

将二进制转换为十进制的例子：`101011 => 43`。

| 二进制 | 101011 |
| :-: | :-: |
| 转换过程 | $1·2^5 + 0·2^4 + 1·2^3 + 0·2^2 + 1·2^1 + 1·2^0$ |
| 十进制 | 43 |

将十进制转换为二进制的例子：`43 => 101011`。

| 43 | $21·2 + 1$ | 1 |
| :-: | :-: | :-: |
| 21 | $10·2 + 1$ | 1 |
| 10 | $5·2 + 0$ | 0 |
| 5 | $2·2 + 1$ | 1 |
| 2 | $1·2 + 0$ | 0 |
| 1 | $0·2 + 1$ | 1 |

# 十进制和十六进制

将十进制转换为十六进制的例子：`101011 => 43`。

| 十六进制 | 4CB2C |
| :-: | :-: |
| 转换过程 | $4·16^4 + 12·16^3 + 11·16^2 + 2·16^1 + 12·16^0$ |
| 十进制 | 314156 |

将十六进制转换为十进制的例子：`314156 => 4CB2C`。

| 314156 | $19634·16 + 12$ | C |
| :-: | :-: | :-: |
| 19634 | $1227·16 + 2$ | 2 |
| 1227 | $76·16 + 11$ | B |
| 76 | $4·16 + 12$ | C |
| 4 | $0·16 + 4$ | 4 |
